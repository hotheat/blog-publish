Numpy 中的广播

发表于 2018-10-11 | 分类于 Python ， Numpy |
| 字数统计: 2.1k

广播指的是指的是不同形状的数组的运算方式，常常在用一个小的矩阵对大的矩阵做一些运算时用到，比如在将向量与矩阵的每一行或每一列做加减运算时，采用循环的方式效率会很慢，Numpy 的广播机制采用向量化的运算，而且在 C 上进行，速度快很多。在标量与向量的运算中，会用到广播，同时也可以利用广播的原理设置数组的值等。

非广播做法

在概念上，我们可以理解成广播把较小的矩阵 b 拉伸成和较大的矩阵 a 相同的形状。

向量与向量

向量与向量的广播最终会形成二维数组，相当于在较大矩阵 a 的每一行加上较小矩阵 b 的对应元素。

向量与二维数组

向量与数组的广播最常见，常常需要在矩阵的每一行或每一列加减某个向量。

  1. 对矩阵的每一列分别加上某个向量
  1. 对矩阵的每一行分别加上某一向量

多维数组的广播

多维数组的广播比较难以理解，但有一点可以记住，经过广播后，产生的数组尺寸和较大的数组尺寸相同。

原理

如果两个数组的后缘维度（从末尾开始算起的维度）的轴长度相符或其中一个后缘维度是 1，则认为它们是广播兼容的。

比如（2，3）的数组可以与（3，）广播，可以与（2，1）的数组广播，但不能与（2，）进行广播。

广播后，数组的尺寸与较大数组的尺寸一样。当两个数组相同时，则优先考虑在行形式表示的向量上加减。

在任何一个维度上，如果一个数组的轴长度为 1，另一个数组的轴长度大于 1，那么在该维度上相当于对第一个数组进行了复制。

应用

np.newaxis

在广播中，常常需要添加一个新轴的方法，reshape 是个办法，但是插入轴需要构造一个表示新形状的元组，Numpy 提供了一种通过全切片的索引机制以及 np.newaxis 属性插入新轴。

两个向量的广播与 np.meshgrid()

虽然二者都返回 (3， 4) 的矩阵，但是两个向量的广播是在较大的数组的每一行加上较小数组的对应元素。

np.meshgrid() 返回的第一个数组是以第一个参数 x 为行，列数为 len(y) 的数组；返回的第二个数组是以第一个参数 x 为列，行数为 len(y) 的矩阵。
