微信文章阅读量和点赞量的爬取

发表于 2019-07-10 | 分类于 使用教程 |
| 字数统计: 392

微信公众号历史文章爬取

微信历史文章标题，时间，url 等信息利用 Chrome 插件 Webscraper 爬取的。

Webscraper 使用方法参考。

用 Webscraper 爬取微信公众号文章参考 https://zhuanlan.zhihu.com/p/42916422。如果出现验证页面，说明微信版本太高，具体参考 http://www.zsxq100.com/make-wechat-content-image/

可以用 json 格式 import sitemap 或 create sitemap。

link 属性如果通过 SelectorLink 拿不到的话，可以 「Element Attribute」方法得到。

微信文章阅读量和点赞量信息在浏览器上是显示不出的，可以通过代理的方式批量获取。

设置代理的方法主要参考文章 https://www.jianshu.com/p/2074f7572694。

需要安装 anyproxy，在 pc 端和移动端安装证书。可以采用 pm2 管理 anyproxy，具体参考

https://www.jianshu.com/p/d978d3b8f2aa。

安装证书步骤参考 anyproxy 官网。

微信爬虫参考 Github https://github.com/lqqyt2423/wechat_spider。

其他注意的几点：

  1. Mongodb 安装参考。

    Mac: https://www.runoob.com/mongodb/mongodb-osx-install.html

  2. 配置 Chromedriver 时，版本需要与 Chrome 浏览器版本相同。

  3. 运行 anyproxy 时，在 8001 端口出现报错 Error: connect ECONNREFUSED 127.0.0.1:80 at TCPConnectWrap.afterConnect [as oncomplete]。暂时忽视这个错误。
